## 서브쿼리(Subquery) 란?
 - SELECT 문 내부에 괄호로 또다시 SELECT 문이 있는 것
 - 어디서 사용되느냐에 따라 명칭이 다르다.
 - FROM절에 사용 : 인라인 뷰(Inline View)
 - SELECT 문에 사용 : 스칼라 서브쿼리(Scala Subquery)
 - WHERE 절에 사용 : 서브쿼리(Subquery)
